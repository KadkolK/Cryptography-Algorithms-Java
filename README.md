# Cryptography-Algorithms-Java
Java Swing application implementing Caesar Cipher and Vernam Cipher | Diploma in Information Technology Project


A Java Swing desktop application implementing **Caesar Cipher** and **Vernam Cipher** encryption and decryption techniques.

This project was developed as part of my **Diploma in Information Technology** coursework. It demonstrates the implementation of classical cryptographic algorithms along with a graphical user interface built using Java Swing.

## Features

* Caesar Cipher Encryption & Decryption
* Vernam Cipher Encryption & Decryption
* Interactive Java Swing GUI
* User-friendly interface for entering plaintext, keys, and viewing encrypted/decrypted output

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)

## Learning Outcomes

* Understanding of classical cryptographic algorithms
* Java GUI development using Swing
* Event-driven programming
* String manipulation and bitwise XOR operations
